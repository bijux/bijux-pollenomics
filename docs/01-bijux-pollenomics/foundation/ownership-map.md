# Ownership Map

The shortest boundary map for contributors:

- source data logic: `bijux_pollenomics.data_downloader`
- candidate ranking logic: `bijux_pollenomics.analysis`
- publication logic: `bijux_pollenomics.reporting`

Use the CLI to print this map quickly:

```bash
bijux-pollenomics ownership-map
bijux-pollenomics ownership-map --json
```
