# Surface Map

`bijux-pollenomics` today is an atlas-builder runtime with reproducible evidence
contracts. It is not yet the full multi-evidence pollenomics engine.

## Runtime Surfaces (Current)

- source collection and normalization
- atlas bundle publication
- country report publication
- candidate ranking artifacts (heuristic)

## Planned Engine Surfaces (Roadmap)

- multi-evidence harmonization runtime
- evidence-aware scoring and interpretation engine
- workflow stage replay and diff execution

## Command Surface

Use the short runtime surface command:

```bash
bijux-pollenomics surface-map
bijux-pollenomics surface-map --json
```
