# RAÄ

RAÄ supplies Sweden-specific archaeology context. It matters because some of
the repository's most detailed regional reading happens in Nordic and Swedish
views, where archaeology density and named context can materially change how a
reader interprets nearby evidence.

## What RAÄ Contributes

- Sweden-scoped archaeology framing
- density and contextual surfaces for Nordic and Swedish outputs
- stronger local archaeological context where the Swedish source coverage is rich

## Where To Inspect It

- `data/raa/normalized/` is the main repository-owned path for this
  Sweden-scoped archaeology family
- RAÄ is valuable precisely because it is Sweden-specific, not because it can
  be stretched into a generic archaeology source

## What RAÄ Does Not Replace

RAÄ is contextual support. It does not replace pollen evidence, sample-level
aDNA evidence, or the direct fieldwork surface.

## Use This Page When

- you want to know why Swedish archaeology context can look denser than other
  geographies
- you need the path behind RAÄ-derived context layers
- you want the reminder that RAÄ helps Sweden and Nordic reading, not general
  world coverage
