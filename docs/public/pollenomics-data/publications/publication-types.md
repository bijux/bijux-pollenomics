# Publication Types

Not every output in the repository plays the same role. Some surfaces are
public-facing evidence views. Others are contextual support, review, or
scaffolding.

## Why Classification Matters

Readers need to know whether they are looking at:

- a public-facing evidence surface
- a contextual support surface
- a review or caveat surface
- a contract or scaffolding surface that exists to keep the public outputs honest

Without that distinction, the report tree looks flatter and more confident than
it really is.
