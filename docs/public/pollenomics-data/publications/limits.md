# Limits

The geography tree is more durable than an older country-first posture, but it
does not magically solve thin evidence. The world surface is still constrained
by current source capture, locality recovery, chronology depth, and coordinate
quality.

## Main Honesty Rules

- world publication is broader than country publication, but it is still not a claim of full completeness
- Europe-plus and Nordic are derived filters, not separate scientific products
- country bundles are narrower views that should never carry facts missing from their parent scope
- blocked, thin, or comparator-only rows should stay visibly qualified instead of being quietly promoted by map convenience

## Why Readers Need This Page

A polished regional map can easily look more complete than the evidence behind
it. This page exists to make the opposite pressure explicit: visual polish must
never outrun review strength.
