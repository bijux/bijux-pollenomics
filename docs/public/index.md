# Public Guide

This part of the site is the public surface for `bijux-pollenomics`. It is
written for readers who want to understand what the repository publishes, where
the visible evidence comes from, how the reports and maps are built, and which
limits still matter before drawing scientific conclusions.

Use these pages if you want the big picture first and the repository details
second.

## What Stays Out Of The Public Guide

The public guide does not teach repository maintenance, release mechanics,
badge synchronization, or workflow administration. Those notes remain in the
repository under `docs/internal/`, but they are not part of the public site.
