# Limits and Audits

The atlas should say clearly when the data foundation is thin, geocoded, or
blocked.

## First Checks

- [`animal_atlas_readiness.md`](../report/animal_atlas_readiness.md)
- [`animal_output_audit.md`](../report/animal_output_audit.md)
- [`animal_country_species_coverage.md`](../report/animal_country_species_coverage.md)

## What These Audits Are For

- readiness: how much curated sample evidence is map-ready by species
- output audit: which species actually reach public outputs
- country coverage: which countries currently have sample-backed animal publication

If these audits show zero country rows or weak geography, that is not a docs
failure. It is the repository telling the truth about the current animal aDNA
foundation.
