# Animal Point Construction

The Nordic atlas should only display points that can be rebuilt from the
checked-in sample, site, chronology, and coordinate evidence tree. The atlas
is a downstream surface, not an authority boundary of its own.

## Construction Rule

One published point should always be defensible through one sample-owned row,
one site-facing justification, one chronology posture, and one coordinate
provenance chain. If any layer is weak, the atlas should demote or exclude the
row rather than promoting it for map convenience.
