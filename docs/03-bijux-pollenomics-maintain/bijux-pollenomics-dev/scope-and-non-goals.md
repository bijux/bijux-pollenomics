# Scope and Non-Goals

The maintainer package is scoped to repository-health behavior, not runtime
scientific behavior.

## In Scope

- API freeze and drift enforcement
- dependency and quality support tooling
- trusted process helpers for maintainer flows
- site asset support used by the docs build

## Out of Scope

- source collection logic
- report publishing logic
- scientific interpretation

## Purpose

This page records the boundary of `bijux-pollenomics-dev`.
