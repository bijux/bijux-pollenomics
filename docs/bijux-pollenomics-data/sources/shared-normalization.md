# Shared Normalization

Even though each source has its own quirks, the repository normalizes them
toward one reviewable shape.

## Shared Expectations

- source outputs remain source-owned
- normalized files are suitable for atlas and report consumption
- country filtering and spatial interpretation stay consistent across layers

## Purpose

This page records the normalization philosophy shared across supported sources.
