# Geographic Limits and Honesty

The geography tree is more durable than the old country-first posture, but it
does not magically solve thin evidence. The world surface is still constrained
by current source capture, locality recovery, chronology depth, and coordinate
quality.

## Main Honesty Rules

- world publication is broader than country publication, but it is still not a claim of full global completeness
- Europe-plus and Nordic are derived filters, not separate scientific products
- country bundles are narrower views that should never carry facts missing from their parent scope
- blocked, thin, or comparator-only animal rows should stay visibly qualified instead of being quietly promoted by map convenience
