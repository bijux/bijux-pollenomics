# Nordic Atlas Limits and Honesty

The atlas should say clearly when the animal foundation is thin, blocked, or
still narrower than the visible map suggests.

## First Checks

- [animal atlas readiness](../../report/animal_atlas_readiness.md)
- [animal output honesty](../../report/animal_output_honesty.md)
- [animal atlas exclusion report](../../report/animal_atlas_exclusion_report.md)
- [animal country species coverage](../../report/animal_country_species_coverage.md)
- [animal output audit](../../report/animal_output_audit.md)

## What These Packets Are For

- readiness: species-by-species status for absent, thin, blocked, and publishable atlas coverage
- honesty: tracked versus mapped versus blocked sample counts in one place
- exclusion: every tracked sample row that stays off the public atlas and the exact blocking reason
- country coverage: which country surfaces still keep sample, site, chronology, and coordinate evidence anchors

If these packets show blocked or unresolved rows, that is not a docs failure. It
is the repository telling the truth about the current animal aDNA foundation.
