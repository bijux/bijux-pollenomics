# Nordic Atlas Filters and Popups

The animal atlas is useful only if readers can inspect the limits of a point,
not just its position.

## Public Animal Controls

- species
- animal scope
- chronology bucket
- Nordic animal leads only
- coordinate confidence

## Popup Contract

Animal popups are citation-first and provenance-first. A useful popup should
show:

- species and scope
- evidence row identifier
- sample and site identifiers
- project accession
- paper DOI
- supplementary source when present
- coordinate basis and confidence
- warnings or caveats
