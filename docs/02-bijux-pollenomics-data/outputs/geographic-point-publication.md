# Geographic Point Publication

Animal map points are published only after sample, site, chronology, and
coordinate evidence have been normalized into one traceable atlas row. The map
is not allowed to invent stronger certainty than those underlying records carry.

## Publication Rule

One visible point should always be explainable through:

- a governing sample lineage
- a site-facing evidence row
- a chronology provenance path
- a coordinate provenance path
- an atlas evidence row that links those surfaces into one published feature

If one of those layers is missing, the map should either downgrade the row or
exclude it from visible publication.
