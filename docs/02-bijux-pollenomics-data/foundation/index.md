# Sample, Site, and Coordinate Contract

The core discipline of the current repository is simple: a published animal
point should be explainable from sample rows, site evidence, and coordinate
provenance, not just from a project list.

## What This Section Settles

- what the repository counts as a sample row
- what turns a paper or supplement into accepted site evidence
- when coordinates are direct, geocoded, projected, or withheld
- which tracked files hold those decisions
