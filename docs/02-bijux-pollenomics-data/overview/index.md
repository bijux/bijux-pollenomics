# Overview

This section introduces the structure of the data reference before it dives
into individual source families or output files.

`bijux-pollenomics` is not a single-dataset repository. It combines pollen and
environmental context, archaeology context, boundary framing, fieldwork
records, and ancient DNA evidence. Those families have different strengths,
different limits, and different paths into public outputs.

## What The Overview Explains

- how the tracked data system is organized
- how animal ancient DNA evidence moves from project records to public outputs
- where to find the main repository-owned data directories
- how this section separates source pages, evidence pages, and output pages
