# Source Comparison

Each supported source contributes a different kind of evidence.

## Comparison

- AADR centers the repository on ancient DNA sample localities
- boundaries supply the country filter and framing logic
- LandClim and Neotoma broaden paleoenvironmental context
- SEAD and RAÄ contribute archaeological context from different national scopes

## Purpose

This page gives readers a quick way to compare why multiple source families are
tracked together.
