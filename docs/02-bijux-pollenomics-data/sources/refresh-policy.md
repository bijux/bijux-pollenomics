# Refresh Policy

Source refreshes should be deliberate and reviewable.

## Policy

- refresh only when there is a clear repository reason
- keep source-specific rewrites explainable in commit history
- regenerate dependent publication outputs when a source refresh changes the
  visible evidence surface
